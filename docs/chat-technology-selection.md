## 方案对比

### 方案A：Daily.co（推荐）
**项目已有依赖**：`@daily-co/daily-js` ✅

| 维度 | 评估 |
|------|------|
| 开发速度 | ⭐⭐⭐⭐⭐ 几小时集成 |
| 功能覆盖 | 视频+语音+文字+屏幕共享 |
| 命理场景匹配 | 极高（八字排盘/塔罗牌面需要视频展示）|
| 成本 | 免费2000分钟/月，初创够用 |
| 数据控制 | 中等（Daily托管，但可导出记录）|
| UI匹配 | 可自定义主题色，或iframe嵌入 |

**实现路径**：
1. 黄总注册 Daily.co → 创建房间 → 给我API key
2. 我集成 Daily Prebuilt UI（嵌入聊天室）
3. 预约时间到，按钮跳转聊天室
4. 聊天历史通过 Daily REST API 获取

## 我的建议

**主推荐：Daily.co**

理由：
1. 项目已有依赖，说明之前考虑过这个方向
2. 命理咨询是"强沟通"场景——八字需要看表情、塔罗需要视频抽牌、风水需要看房间布局
3. 文字聊天只是Daily的功能之一，视频才是命理咨询的核心体验
4. 2000分钟免费额度 = 约60次30分钟咨询，初创够用3-6个月
5. 今晚几小时就能出可交互demo

**备选：Supabase Realtime（纯文字）**
如果黄总认为文字足够，可以快速切换。但建议先用Daily验证视频+聊天的完整体验。
